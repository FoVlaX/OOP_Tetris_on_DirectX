Чтобы скомпилировать проект необходимо скачать 
DirectX SDK 11 https://www.microsoft.com/en-us/download/details.aspx?id=6812
В свойствах проекта указать
допольнительные каталоги включамых файлов: путь до \Microsoft DirectX SDK (June 2010)\Include
допольнительные каталоги библиотек: путь до Microsoft DirectX SDK (June 2010)\Lib\x86(или х64 в зависимости под какую платформу компилировать)
в release есть готовый скомпилированный файл и используемые им файлы, чтобы запустить его необходима установлнная версия DirectX 9.* и выше
